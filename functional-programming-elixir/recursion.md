## Fibonacci Numbers
```
Fibonacci(0) = 0
Fibonacci(1) = 0
Fibonacci(2) = 1
Fibonacci(n) = Fibonacci(n-1) + Fibonacci(n-2)  
```

### Sample Input
```
3
```
### Sample Output
```
1
```
